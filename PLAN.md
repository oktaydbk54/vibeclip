# shorts-mcp — Uzun Video → Shorts/Reels MCP Pipeline

Uzun format videoları (podcast, röportaj, eğitim, konuşma ağırlıklı içerik) otomatik
olarak analiz edip viral kısa kliplere bölen, dikey formata getiren ve altyazı yakan
bir MCP server. Claude veya herhangi bir MCP-uyumlu agent bu tool'ları çağırır.

## Mimari (özet)

```
Video
  → transcribe   (WhisperX: kelime-bazlı zaman damgalı transkript)
  → find_highlights (DeepSeek: transkriptten en iyi anları seçer)
  → cut_clip     (ffmpeg: -c copy ile anında kesim)
  → reframe_vertical (9:16, yüz-takipli crop)
  → burn_subtitles   (kelime-senkron zıplayan altyazı)
  → Shorts/Reels
```

DeepSeek **videoyu görmez, transkripti okur** — bu yüzden konuşma ağırlıklı içerikte
mükemmel, saf görsel/aksiyon highlight'ta (spor, gameplay) sınırlı.

## Teknoloji

| Katman | Seçim | Not |
|---|---|---|
| Dil / MCP | Python 3.12 + `mcp` (FastMCP) | Whisper/ffmpeg ekosistemi |
| STT | WhisperX (faster-whisper int8) | kelime zamanı + VAD; Mac'te mlx-whisper alternatifi |
| Beyin | DeepSeek API (deepseek-chat) | ucuz, JSON çıktı |
| Kesim/render | ffmpeg 8.x | `-c copy`, VideoToolbox GPU encode (Apple Silicon) |
| Dikey crop | OpenCV/MediaPipe yüz takibi | Faz 4 |
| Altyazı | ffmpeg `ass` | kelime-senkron |

## MCP Tool'ları

1. `transcribe(video_path)` → `{segments, words[], language}` (cache'li)
2. `find_highlights(video_path, platform, count, max_duration)` → `[{start, end, title, hook, score}]`
3. `cut_clip(video_path, start, end)` → ham klip yolu
4. `reframe_vertical(clip_path)` → 9:16 dikey klip
5. `burn_subtitles(clip_path, words)` → altyazılı klip
6. `make_short(video_path, ...)` → uçtan uca zincir (tek komut)
