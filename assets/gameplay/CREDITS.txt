Split-screen "gameplay background" packs — footage credits & licenses
======================================================================

Every clip in this folder is copyright-cleared for reuse. The split-screen
stage (pipeline/splitscreen.py) uses these only as the muted BOTTOM region
of a clip; their own audio is never used.

Keep this file in sync with assets/gameplay/. If a clip's license requires
attribution (CC BY), that attribution must remain visible wherever the
rendered output is published — same rule as the landing demo footage.
